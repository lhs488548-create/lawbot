## 3. 판례 추출 계획 (2-phase, 판례에서만 칼같이)

폴더가 `{사건종류}/{등급}/{법원}_{선고일}_{사건번호}.md`로 **주제 라벨 없음** → 내용기반 매칭만 가능.
실측 모수: 형사 대법원 15,060·하급심 6,607 / 일반행정 대법원 13,609·하급심 31,495 / 민사 대법원 28,528·하급심 13,604.

- **Phase-1 (고precision, 1차 빌드)**: 형사+일반행정 **대법원만** → `## 참조조문` 화이트리스트 법령명 **정확매칭**(공백·괄호·'구 의료법(개정전)'·약칭 정규화). 추정 채택 **1,500~4,000건**.
- **Phase-2 (품질측정 후 확장)**: 사건명/판시사항 의료 키워드 union(무면허·면허정지·환자유인·리베이트·요양급여 부당청구·진료기록 거짓작성 …) + 동명이의 본문 2차검증 + 하급심.
- **필수 메타**: 채택마다 `match_rule`·`matched_term`·사건번호·법원·등급·선고일 → 사후 false-positive 감사 + RAG citation.
- 실측 보유율: 판시사항 62.0% · 참조조문 61.3% · 판결요지 55.8% (강신호 동작 모수 충분).

## 4. "의료관련" 폴더 구조 + 추출 방법

> ⭐ **원천 6GB는 복사하지 않는다.** manifest로 가리키고 파서가 스트리밍 → 멱등·증분·감사.
> "의료관련 폴더에 관련데이터 모으기"는 **파싱된 JSONL 형태**로 충족(raw .md 복사 아님).

```
artifacts/의료관련/
├─ manifest/                      ← 버전관리·멱등 핵심(YAML)
│   ├─ statutes_whitelist.yaml    (Tier1+2 폴더명, NFC 키)
│   ├─ admrule_targets.yaml       (부처경로+규칙명 키워드)
│   ├─ admrule_denylist.yaml      (국립병원예규·직제·계약·장학)
│   └─ precedent_match_rules.yaml (참조조문 정규화·키워드)
├─ docs/  {국가법령.jsonl, 행정규칙.jsonl, 판례.jsonl}   ← 필터링된 Document
├─ chunks.jsonl + vectors.jsonl                          ← chunk + cached_embed
└─ _audit/ {match_log.csv, absent_null.csv, dedup_log.csv}
```

**추출 파이프라인** (복사 X, 필터-파서):
1. 국가법령 → whitelist 폴더의 `법률/시행령/시행규칙.md`만 `parse_statute.parse_file`
2. 행정규칙 → `parse_admrule.parse_all()` + 부처스코핑·키워드·denylist·trust_grade 분기
3. 판례 → `parse_precedent.parse_all()` 스트리밍 + match_rules(Phase-1) + 메타기록
- 검증: 화이트리스트 폴더 실재 사전 assert(누락=빌드실패) · trust A/B 카운트 · 판례 top-k 노이즈율 샘플(Phase-2 게이트) · **absent_null.csv로 부재항목 명시(커버리지 착시 차단)** · content-hash 캐시로 재임베딩 0

## 6. FAISS 권고

- **`IndexFlatIP`(Inner Product, 완전탐색) 확정.** OpenAI 1536d 벡터는 L2 정규화 → 내적=코사인 동치. 의료 서브코퍼스 규모(수만~17만)에서 brute-force도 수ms~수십ms, **recall=100%** → Citation 검증·정밀검색 이상적. 학습·파라미터 불필요, 멱등 재현.
- 정규화 전제만 지키면 됨(미정규화 시 내적≠코사인). 수백만으로 커지면 그때 `IndexHNSWFlat(IP)`.

## 7. 코퍼스 부재 4종 (NULL 명시 + fallback)

실측상 원천데이터에 **본문이 없는** 항목 → `absent_null.csv`에 기록 + 조문 fallback으로 **환각 차단**:
1. **의료광고 자율심의기준** (의협·치협·한의협 내규 = 정부고시 아님) → 의료법 §56~57의3 + 시행령
2. **비대면진료 시범사업 지침** (복지부 트리 무매칭) → 의료법 §34 fallback
3. **의료기기 광고심의 위탁 고시** (의약품만 존재) → 의료기기법 §24~26 시행규칙
4. **비의료 건강관리서비스 가이드라인** (부재) → 의료법 §27 무면허 + 유권해석
- 헌재 결정례·법령해석례·행정심판 재결례 **0건** (의료광고 사전심의 위헌결정 부재) — 판례 RAG 한계 명시.
